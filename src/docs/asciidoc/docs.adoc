=== HTTP status codes

|===
| 상태 코드 | 설명

| `200 OK`
| 성공

| `201 Created`
| 리소스 생성

| `204 NO_CONTENT`
| 성공 후 반환 값 없음

| `400 Bad Request`
| 잘못된 요청

| `401 Unauthorized`
| 비인증 상태

| `403 Forbidden`
| 권한 거부

| `404 Not Found`
| 존재하지 않는 요청 리소스

| `500 Internal Server Error`
| 서버 에러
|===

===== rainType
|===
| 값 | 설명

| `CLEAR`
| 맑음

| `PARTLY_CLOUDY`
| 구름 많음

| `CLOUDY`
| 흐림
|===

===== skyStatus
|===
| 값 | 설명

| `NONE`
| 없음

| `RAIN`
| 비

| `RAIN_AND_SNOW`
| 비/눈

| `SNOW`
| 눈

| `SHOWER`
| 소나기

| `RAIN_DROP`
| 빗방울

| `RAIN_AND_SNOW_FLURRIES`
| 빗방울눈날림

| `SNOW_FLURRIES`
| 눈날림
|===
